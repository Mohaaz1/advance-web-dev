PHP + MySQL vs MEVN Stack

The lesson uses PHP and MySQL — our equivalent uses Node.js and MongoDB

MongoDB Mongoose Connection
Concept PHP + MySQL MEVN Equivalent
Database type Relational (MySQL) NoSQL (MongoDB)
Connection method mysqli_connect() mongoose.connect()
Connection string localhost, root, "", db mongodb://127.0.0.1:27017/db
Success message echo "Connected" console.log("Connected")
Tool to manage DB phpMyAdmin MongoDB Compass / mongosh

Database Connection Code

The connection is made in index.js when the server starts

mongoose.connect index.js
// index.js - equivalent of PHP mysqli_connect()
const mongoose = require('mongoose');

mongoose.connect('mongodb://127.0.0.1:27017/bit3208_week3')
  .then(() => console.log('Connected to MongoDB'))
  .catch(err => console.log('Connection failed:', err));
