Welcome Message Page

Calls GET /welcome on the Express server - equivalent to PHP echo

GET Route req.query res.json
// index.js - Node.js equivalent of PHP welcome page
app.get('/welcome', (req, res) => {
  const name = req.query.name || 'Student';
  res.json({ message: 'Welcome, ' + name });
});

Simple Form Processor

Submits form data via POST to the Express server - equivalent to PHP $_POST

POST Route req.body Form Handling
// index.js - Node.js equivalent of PHP form processor
app.post('/submit', (req, res) => {
  const { name, email } = req.body;
  if (!name || !email) return res.json({ success: false });
  res.json({ success: true, message: 'Hello ' + name });
});
